# v0.1.0

Primera version publica del controlador de escritorio para switches Tuya/Tuya Smart.

Contenido:

- `Tuya Desk Controller_0.1.0_x64-setup.exe`: instalador NSIS para Windows x64.
- `SHA256SUMS.txt`: hash SHA-256 del instalador.

Notas:

- Build generada con `npm run tauri build`.
- Frontend en Vite + TypeScript.
- Backend en Tauri + Rust.
